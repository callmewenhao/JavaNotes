
#### 练习

```java
int i = 1;
i = i++; // (1)temp=i;(2)i=i+1;(3)i=temp;
System.out.println(i); // i=1

int i = 1;
i = ++i; // (1)i=i+1;(2)temp=i;(3)i=temp;
System.out.println(i); // i=2
```
