# 类与对象（中级）

## idea 常用快捷键

- ctrl + Y：delete Line

- ctrl + D：copy line

- alt+ /：补全代码

- alt + enter：自动导入类

- ctrl + alt+ L：格式化代码

- ctrl + shift+ F10：快速运行

- 生成构造器等：alt + insert

- 查看类的继承关系：ctrl + h

- 查看方法的实现：ctrl + b

- 自动分配变量名：.var + 回车

使用 idea 模板提升代码效率，查看模板：`file->Settings->Editor->Live Templates->java`
